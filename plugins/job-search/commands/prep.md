# Prep Command

> If you see unfamiliar placeholders or need to check which tools are connected, see [CONNECTORS.md](../CONNECTORS.md).

Prepare comprehensively for an interview.

## Instructions

### 1. Load Context

Check `memory/companies/[company].md` for existing research. If missing or stale, run research first.
Read `JOBSEARCH.md` for profile and experience.
Read `${CLAUDE_PLUGIN_ROOT}/skills/job-search/references/interview-prep.md` for the full framework.

### 2. Research Interview Process

Search for:
- "[company] interview questions" (Glassdoor, AmbitionBox, Blind)
- "[company] [role] interview process" (rounds, format)
- "[company] culture fit interview"
- Interviewer backgrounds if names provided

### 3. Build Prep Package

Generate the full prep document:
- **Interview Format** — expected rounds and format
- **Behavioral Questions** with STAR answers mapped to user's actual experience
- **Role-Specific Questions** with framework-based answers
- **Culture Fit Questions** with what-they're-really-asking context
- **Questions to Ask** — stage-appropriate, showing research
- **Interviewer Profiles** — if names known
- **Company Cheat Sheet** — key facts for quick reference
- **Day-Of Checklist** — logistics, materials, mindset

### 4. Save and Update Memory

Save prep package to `memory/applications/[company]-[role]/interview_prep.md`.
Update `CLAUDE.md` with upcoming interview details.
Log prep completion in `memory/analytics/interviews.md`.
